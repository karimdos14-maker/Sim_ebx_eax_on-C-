 Simple ASM Emulator
This is a lightweight Assembly registers and arithmetic emulator written in C#.

 Key Features
*   Simulates EAX, EBX, ECX, EDX registers.
*   Supports ADD, SUB, MUL, DIV operations.
*   Self-made without any tutorials.
*   **Refactoring:** Optimized from 100 lines of code down to 31.

 Why I made this
I am 11 years old, and I wanted to understand how CPU registers work under the hood.
